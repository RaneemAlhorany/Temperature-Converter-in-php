# Temperature-Converter-in-php

It's stage 1 in your PHP language learning journey

project (4) : Temperature-Converter-in-php 

Create a simple temperature converter that:
     User inputs:
     
        A temperature value
        Selects the conversion type:
        Celsius → Fahrenheit
        Fahrenheit → Celsius

Output:

    The converted temperature displayed on the page.

➡️ Requirements:

    Use an HTML form
    Use GET or POST
    Use conditions (if / switch)
    Use simple math formulas

    
Conversion Formulas:
    
    C → F:
    > F = (C × 9/5) + 32
    
    F → C:
    > C = (F - 32) × 5/9
